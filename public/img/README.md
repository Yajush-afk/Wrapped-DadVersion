# Photo guide

Drop your photos in this folder using the filenames below. The website shows designed placeholders until each file exists, so you can add them incrementally.

| Filename | Used for |
| --- | --- |
| `cover.jpg` | Opening hero |
| `montage-1.jpg` | MVP collage |
| `montage-2.jpg` | MVP collage |
| `montage-3.jpg` | MVP collage |
| `driving.jpg` | Family kilometres card |
| `habit.jpg` | Problems/chai cards |
| `quote.jpg` | Advice card |
| `family.jpg` | Support and top-role cards |
| `memory-1.jpg` … `memory-8.jpg` | Constellation memories |

Landscape and portrait photos both work. Use JPG, WebP, or PNG files; if you change an extension, update the matching path in `src/content/dadWrapped.ts`.

For good loading performance, aim for 1200–1800 pixels on the longest edge and under roughly 600 KB per image.
